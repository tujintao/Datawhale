###  Q & A

###  1. 为什么要扩容？

静态数组的空间是有限的 ，每次插入空间都会减少，动态扩容一方面扩大了数组的容量，一边面减小的溢出的风险

### 2.扩容多大合适呢?

### 3.为什么要缩容

算是节省空间的一种办法，数组中的空间利用率本身很低，比如某些格外关注空间利用率的场合。
